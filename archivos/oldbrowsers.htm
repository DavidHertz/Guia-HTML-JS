A prueba de veteranos

Ya lo sabes: JavaScript no funciona igual en las diferentes ediciones de los dos navegadores más populares: Netscape Navigator y Explorer. Aunque es muy difícil que un usuario siga usando las primeras versiones de estos browsers, hay que estar atentos y prevenir cualquier error a los internautas desactualizados.

Todos los navegadores omiten las etiquetas desconocidas, por lo tanto, aquellos que no "entiendan" JavaScript ignorarán lo que encierra el tag <SCRIPT>. Pero para ocultar el resto del código y evitar que aparezca en la pantalla del navegante, conviene guardarlo entre los símbolos que el HTML usa para los comentarios: , como podrás ver si cliqueas sobre la figura.


Ampliar

Analiza el ejemplo. Los navegadores que soporten JavaScript leerán el código encerrado en el tag <SCRIPT> y dejarán de lado la línea que comienza con la doble barra (//), signo para los comentarios propio de este lenguaje. Es muy importante el uso de la "//" para evitar un fallo en el programa.

De todos modos, ocultar el código es una buena forma de evitar problemas con esos "veteranos" que prefieren seguir fieles a sus viejos y queridos browsers. Pero ¿qué sucede con aquellos navegantes despistados que poseen un navegador capaz de leer JavaScript, pero que tienen la opción desactivada?

En caso contrario, se ejecutarán normalmente los procedimientos creados dentro de la etiqueta <SCRIPT>.
