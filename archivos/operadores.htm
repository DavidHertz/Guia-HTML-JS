Operadores

En uno de los ejemplos anteriores viste como se formaba una expresión al unir el contenido de variables con otra información. Esa tarea se realiza mediante operadores, que sirven para concatenar y realizar operaciones matemáticas o lógicas con los datos.

Aunque JavaScript dispone de muchos tipos de operadores, a los fines de esta guia basta explicar los más útiles:

A) Operadores Aritméticos

B) Operadores de Comparación

C) Operadores Lógicos

 Operadores Aritméticos

Son los que permiten hacer las tareas matemáticas más comunes. Son imprescindibles, por ejemplo, para crear una calculadora en tu sitio. Para ver una lista de operadores y ejemplo de cada uno, cliquea aqui.

Operadores de Comparación

Quien acostumbra utilizar fórmulas condicionales en el programa de planillas de cálculo Excel o consultas en el de bases de datos Access ya está familiarizado con estos operadores. Se usan, sobre todo, para establecer requisitos para que se ejecute una función. ¿Un caso? mostrar en una tabla todos los números de una lista que sean mayores que 5. Los operadores de comparación más comunes están detallados en este cuadro.

Operadores Lógicos

Usados en conjunto con los operadores de comparación, permiten establecer expresiones lógicas más complejas, por ejemplo, para saber si se deben cumplir obligatoriamente dos condiciones (Y) o sólo alguna de las que se establezca (O) al momento de realizar una operación. Para ver una lista de operadores lógicos cliquea aquí.
