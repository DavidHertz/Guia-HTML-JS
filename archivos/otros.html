A continuacion te voy a explicar dos etiquetas que son muy importantes, y agradables:

La primera es <MARQUEE>...</MARQUEE>, Esta etiqueta se utiliza para insertar un texto movil.

EJEMPLO:

<MARQUEE>Este es un texto movil</MARQUEE>

El cual se vera asi:

Este es un texto movil

Esto solo podra ser visualizado por los que tengan el Explorador 3.0 o superior.

Para resaltar más este efecto, puedemos incorporar un color de fondo con el atributo BGCOLOR.

<MARQUEE BGCOLOR=blue> ja-ja-ja-ja </MARQUEE>

El cual se veria asi:

ja-ja-ja-ja

Tambien puede ir acompañados de los atributos HEIGHT (alto) y WIDTH (ancho).Estos deben ir seguido de un valor numerico el cual se mide en pixeles

<MARQUEE BGCOLOR=red HEIGHT=25 WIDTH=50%> cogelo</MARQUEE>

Se veria asi:

David K

Tambien podemos controlar el numero de veces que queremos que salga el mensaje, y el sentido(de izquierda a derecha o al contrario), el primero se logra con el atributo LOOP, y el segundo con DIRECTION, el cual tiene dos valores RIGHT o LEFT, su valor por defecto es LEFT.

EJEMPLO:

<MARQUEE LOOP=1 DIRECTION=RIGHT > Si no lo viste, actualiza la pagina y esta atento </MARQUEE>

Si no lo viste, actualiza la pagina y esta atendo

Tambien puedes utilizar las etiquetas vistas anteriormente, es decir, pudes cambiarle el color a la letra, puedes ponerla en cursiva, etc.

EJEMPLO:

ja-ja-ja-ja

La segunda etiqueta es parecida a la anterior, solo que el texto encerrada comienza a parpadear, es propio del NETSCAPE.

EJEMPLO:

<BLINK>ta-ta-ta-ta </BLINK>

ta-ta-ta-ta

Como estas etiquetas son propia de cada navegador, debes tener cuidado a la hora de utilizarlo porque no sabemos que navegador esta utilizando nuestro visitante, lo cual puede ser muy perjudicial para nuestra pagina.
Te aconsejo que si los vas a utilizar los coloques en la misma linea, con el objetivo de que si no reconoce el uno reconocera el otro.

•Otra marquesina muy util es la siguiente que es capaz de visualizarse en ambas versiones de Internet Explorer y Netscape, su funcion es rebotar en las pareredes de nuestra pagina web

<marquee behavior="alternate">¿Como haz estado?</marquee>

¿Como haz estado?


•Otro tag muy bueno es esta que sirve para darle cierta velocidad a nuestra marquesina, el codigo es el sigueinte:

<marquee scrollamount=50>Hola Amigo!</marquee>

Hola Amigo!

Dependiendo de la velocidad que queramos aplicar se le asignara un numero que va desde 1 a 100.


•Este otro ejemplo sirve para darle un efecto de "paradas"

<marquee scrolldelay=500 scrollamount=100>Hola Amigo!</marquee>

Hola Amigo!

En ese ejemplo scrolldelay tiene un valor que es de 1 a 1000 donde 1 es la velocidad maxima y 1000 la velocidad minima.
