El ejemplo es un poco tonto, y hasta estarás pensando que podrías lograr algo igual sólo con HTML y menos trabajo. Pero su sencillez ilustra la esencia de JavaScript. Ya conoces la etiqueta <SCRIPT>, pero ¿cómo se traduce la sentencia que se encierra en ella?
JavaScript se basa en objetos, los cuales tienen propiedades, métodos y parámetros. Sigue leyendo esta guia y proximamente sabras más sobre ellos, para ejemplificarlo fácilmente, podríamos decir que el objeto "lápiz" tiene, entre sus muchas propiedades, su "color". Y como método, la acción de "escribir" cierta frase. Con esta idea, la sintaxis básica para objetos en JavaScript se resume así:

objeto.propiedad
objeto.método(parámetro)

Siguiendo este razonamiento, con el código escrito anteriormente lográs que en la página web (objeto document) se escriba (método write) la frase que encerramos entre paréntesis (un parámetro).
Y para terminar la orden, en lugar de utilizar el "punto y aparte" del lenguaje escrito, se usa el símbolo punto y coma (;). Algo muy importante, ya que en JavaScript ese signo de puntuación indica el final de cada instrucción.

Sencillo, ¿no? Pero a no descuidarse, porque si deseas ser un experto en JavaScript debes saber que este primer programa todavía no tiene un aspecto profesional.
