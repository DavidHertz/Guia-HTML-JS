Abre una ventana en tu sitio

Objetos, propiedades y métodos: ya leíste suficiente sobre ellos en partes atras de esta guia. Ahora llegó el momento de trabajar con ellos. Pero quizas, antes de hacerlo necesites refrescar alguna idea, cliquea aquí para leer el capítulo de este curso donde se explican esos conceptos.

Para este lenguaje de programación, todo lo que hay en tu navegador es un objeto, que puede ser modificado a través de propiedades y métodos. Dentro de la jerarquía en la que JavaScript ordena a los objetos (un click aquí te la muestra), el más importante se llama Window y representa la ventana de tu navegador.

Las propiedades y métodos del objeto Window, ayudan a crear ventanas flotantes (o "Pop-Ups"), elegir su apariencia y hasta mostrar diferentes cuadros de diálogo con el usuario.

Las propiedades de una ventana

¿Para que usar las propiedades? Recuerda que son atributos que definen a un elemento. En este caso, el objeto Window tiene muchas propiedades, aunque aquí sólo encontrarás las que son imprescindibles para dar forma a una ventana.


Ampliar

Esta es la lista:

defaultStatus:

Determinar el mensaje que aparecerá en la barra de estado del navegador.

Sintaxis: window.defaultStatus="mensaje"


document:

Refiere a la página actual que exhibe la ventana. Document es un objeto y, como tal, tiene sus propiedades y métodos, que conocerás cuando se hable exclusivamente de este objeto.

Sintaxis: window.document


history:

Listado (o "Array") que contiene las direcciones web visitadas por el usuario. También es un objeto con propiedades y métodos que permiten acceder a esas URLs almacenadas.

Sintaxis: window.history


location:

Refiere a la actual dirección web del navegador o sirve para definir un destino específico, ya sea un sitio o un archivo de la PC.

Sintaxis: window.location="http://...." ó "file:///C:/Carpeta"


locationbar, menubar, scrollbars, statusbar y toolbar:

Representan la barra de dirección, de menú, de desplazamiento, de estado y de herramientas, respectivamente. En todas se aplica la propiedad "visible" para definir su presencia en la ventana colocando el valor "true" o "false".

Sintaxis: window.menubar.visible="false"


Status:

Sirve para definir un mensaje momentáneo o animado (Scroll) en la barra de estado del Browser. Recuerda que esta es una propiedad diferente de defaultStatus y de statusBar. Se emplea, por ejemplo, para mostrar una breve explicación de la función que cumple un botón, en el momento en el que el usuario pasa el cursor sobre él.

Sintaxis: window.status="mensaje"

El método y su discurso

Los métodos son las funciones primordiales que contiene cada objeto. Para abandonar los tecnicismos, podría decirse que es lo que está dado en la naturaleza de un cierto objeto.Así si un balón fuera un objeto, su método sería "rodar", una propiedad que tiene naturalmente. Volviendo a JavaScript, aqui van los métodos más útiles de Window.

alert:

Muestra un cuadro de advertencia que avisa al usuario sobre cierta situación. Por ejemplo, un error en la página. Para ver un cuadro tipo alert, cliquea aquí.

Sintaxis: window.alert("mensaje")

close:

Cierra ventanas que hayan sido creadas por el método open. Si se usa para cerrar ventanas de otro tipo, el usuario deberá responder un cuadro de interrogación para confirmar la acción, salvo que la ventana activa tenga un solo documento en su historial.

Sintaxis: window.close()

confirm:

Muestra un cuadro de interrogación para que el navegante confirme una acción mediante los botones Aceptar o Cancelar. Estos botones tienen el valor "True" y "False" respectivamente, y para definir la elección que hizo el usuario se programan con una estructura de control, que se verá más adelante. Para ver un cuadro del tipo confirm, haz click aquí.

Sintaxis: window.confirm("mensaje")

open:

Abre una nueva ventana en el navegador. En su sintaxis hay que aclarar una URL de destino (que puede ser un sitio o un documento alojando en el mismo servidor), un nombre para la nueva ventana y las características que tendrá.

Sintaxis: window.open(URL,nombre,características)

prompt:

Crea un cuadro de diálogo para que el usuario brinde un dato , por ejemplo, una contraseña para ingresar a cierta página. Tiene un parámetro para definir el mensaje a mostrar y otro, opcional, para mostrar una información dentro del cuadro de texto de la ventana. Al igual que los cuadros tipo confirm, se usajunto a una estructura de control para determinar la acción del usuario. Para ver un ejemplo de prompt presiona aquí.

Sintaxis: window.prompt("mensaje", "sugerencia")

setTimeout:

Este método llama a una función después de una determinada cantidad de milisegundos. Se usa para realizar una operación que deba ejecutarse a cierta velocidad.

Sintaxis: window.setTimeout=("nombre_función", tiempo)

back, forward, home, stop:

Estos métodos realizan las funciones de navegación básicas de cualquier browser, como retroceder una página, avanzar, ir a la homepage o detener la descarga pero no son reconocidos por el Internet Explorer.

Sintaxis: window.back()

Ampliar

Como este script requiere de muchas acciones, ellas están reunidas en la función AbrirVentana.

Allí verás la variable "miventana" capaz de almacenar la tarea de abrir una nueva ventana usando el método open. Pero seguramente te preguntarás qué son todos esos códigos entre sus paréntesis.

Las dobles comillas ("") definen la aparición de una página en blanco. Si quisieras que la ventana muestre determinado sitio web, tendrías que escribir la URL indicada. "Ventana" es el nombre del Pop-Up, y el resto del código define la apariencia que tendrá. En este caso, oculta la barra de herramientas, los menúes y define su tamaño.

A cualquier ventana se le puede dar la forma que uno desee cambiando estas características.

directories= yes / no Barra de Vínculos
location= yes / no Barra de Dirección
menubar= yes / no Barra de Menú
toolbar= yes / no Barra de Herramientas
scrollbars= yes / no Barras de desplazamiento
resizable= yes / no Posibilidad de redimensionar la ventana
status= yes / no Barra de Estado
height= número en pixels Alto de la ventana
width= número en pixels Ancho de la ventana

El uso de estos parámetros del método open es opcional, pero si no los declaras se abrirá una ventana con todos los elementos visibles. Otro dato: no hace falta que agregar "yes" por cada objeto a mostrar, ya que la sola mención de su nombre cumple la misma función.

Volviendo al código de nuestro script, una vez definida la apariencia del Pop-Up, se usa la sentencia document.write para crear, primero, un mensaje, y luego, un vínculo que al ser cliqueado cerrará la ventana por medio del método close().
